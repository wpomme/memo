# Vim
## help の読み方
- help を全画面で見る
```
:h | only
```

- ユーザーマニュアルの目次を開く
```
:h user-manual
```

- 指定した項目を開く
```
:h usr_06.txt
```

## 組み込みファイラ netrw
- 起動
```
:Ex
```

- 表示モード切り替え
`i`

thin -> long -> wide -> tree
* このdotfilesではデフォルトの表示モードをtree にしてある

- 新しいタブで開く
`t`

## 閲覧モードなど
- 編集不許可の`-M` オプションを付けると便利。neovim でも同様。余計なキーを押したときに編集が不可能になる。
```
nvim -M error.log
```

- 読み取り専用にする場合は`-R` オプションを付けるなど。
```
vim -R error.log
```
